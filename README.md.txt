Sistema Hotel - CRUD en Java

Descripción:
Este proyecto permite gestionar habitaciones de un hotel utilizando Java y MySQL con JDBC.

Funcionalidades:
- Insertar habitaciones
- Listar habitaciones
- Actualizar habitaciones
- Eliminar habitaciones

Tecnologías utilizadas:
- Java
- MySQL
- JDBC
